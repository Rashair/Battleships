# Battleships
Simple simulation of Battleships game ([Wiki](https://en.wikipedia.org/wiki/Battleship_(game)))

## Assumptions
1. Default settings:

| No.  | Class of ship   | Size |
| ---- | ----------------- | ------|
| 1	| Carrier	      |5 |
| 2	| Battleship      |4 |
| 3	| Cruiser	      |3 |
| 4	| Submarine	      |3 |
| 5	| Destroyer	      |2 |

2. Ships can touch each other, but they can't occupy the same grid space
